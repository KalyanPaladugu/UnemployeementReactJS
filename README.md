### Media Queries

+ `@media` rule
```
@media screen and (max-width:320px) and (min-width:200px)
{
<!-- css styles -->
}
```

+ Extra small devices
`@media only screen and (max-width:600px){
 css code
}
`
+ small devices
`@media only screen and (min-width:600px){

}
`

`
+ Large devices
